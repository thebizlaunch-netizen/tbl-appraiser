## First launch on macOS

Current builds are unsigned (Apple notarisation in progress). To open:

1. Download the correct `.dmg` for your Mac (`-arm64` for Apple Silicon, no-suffix for Intel).
2. Open the `.dmg` and drag `TBL Appraiser.app` to `Applications`.
3. Right-click the app → **Open** → **Open** in the confirmation dialog. Just the first time.
4. From then on, launches normally from Launchpad / Spotlight.
